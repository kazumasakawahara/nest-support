# 相続カルキュレーター Agent Skill

## 📋 概要

このスキルは、日本の民法に基づいて法定相続人と相続分を自動計算します。相続に関する質問を受けた際、このスキルを使用して正確な相続分を計算してください。

## ✨ 主な機能

✅ 配偶者の相続権の確認  
✅ 血族相続人の確定（第1順位〜第3順位）  
✅ 代襲相続の判定（子は世代を超えて可能、兄弟姉妹は一代限り）  
✅ 相続放棄の考慮  
✅ 同時死亡の推定の適用（民法32条の2）  
✅ 法定相続分の自動計算  
✅ 半血の兄弟姉妹への対応（相続分は全血の1/2）  

## 🎯 使用タイミング

以下のような質問を受けた際に、このスキルを使用してください：

- 「相続分を計算してください」
- 「法定相続人は誰ですか」
- 「代襲相続はどうなりますか」
- 「配偶者と子がいる場合の相続分は？」
- 「兄弟姉妹が相続人になる場合は？」
- その他、相続に関する計算が必要な質問

## 📚 相続の基礎知識

### 相続順位

1. **配偶者**: 常に相続人となります（生存している場合）
2. **第1順位**: 子（代襲相続あり、世代を超えて可能）
3. **第2順位**: 直系尊属（父母、祖父母など。代襲相続なし）
4. **第3順位**: 兄弟姉妹（代襲相続は一代限り=甥・姪まで）

### 法定相続分

- **配偶者 + 子**: 配偶者 1/2、子全体 1/2
- **配偶者 + 直系尊属**: 配偶者 2/3、直系尊属全体 1/3
- **配偶者 + 兄弟姉妹**: 配偶者 3/4、兄弟姉妹全体 1/4
- **血族相続人のみ**: 該当順位で100%

### 特殊なケース

- **代襲相続**: 本来の相続人が既に死亡している場合、その子が代わりに相続
- **相続放棄**: 相続放棄した人は初めから相続人ではなかったものとみなされる
- **同時死亡の推定**: 災害等で同時に死亡した場合、その人たち同士では相続が発生しない
- **半血の兄弟姉妹**: 片親のみ同じ兄弟姉妹の相続分は、全血（両親とも同じ）の1/2

## 🚀 使用方法

### ステップ1: ユーザーからの情報収集

ユーザーから以下の情報を聞き取ります。不足している情報がある場合は質問してください：

1. **被相続人の氏名**
2. **同時死亡の有無**（災害・事故等で親族と同時に死亡したか）
3. **配偶者の情報**
4. **子の情報**（複数いる場合は全員）
5. **直系尊属の情報**（父母、祖父母など）
6. **兄弟姉妹の情報**

### ステップ2: スクリプトの実行

```bash
python3 claude-skills/inheritance-calculator/inheritance_calculator.py --pretty /tmp/inheritance_input.json
```

### ステップ3: 結果の解釈と提示

計算結果をユーザーに分かりやすく説明します。

**必ず追加する注意事項**: 「このツールは参考情報を提供するものであり、法的助言ではありません。実際の相続では必ず専門家（弁護士、司法書士等）にご相談ください。」

## ⚠️ 制限事項

このスキルでは以下の計算は行えません：

- 遺言による指定相続分
- 特別受益の持ち戻し計算
- 寄与分の計算
- 遺留分の計算
- 具体的な相続財産の評価
- 相続税の計算

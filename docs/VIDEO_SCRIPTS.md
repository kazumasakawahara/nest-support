# 動画チュートリアル 台本集

> 5分×5本の短尺動画セット。画面収録 + ナレーションを想定。
> 撮影前にデモデータを投入しておくこと: `./installer/load-demo-data.sh`

---

## 動画1: インストールと初期設定（5分）

**タイトル**: 「nest-support を10分で動かす」

### シーン構成

| 時間 | 画面 | ナレーション |
|------|------|-------------|
| 0:00-0:30 | タイトル画面 | 「nest-support は、障害のある方の支援情報を記録・共有するためのデータベースシステムです。このビデオでは、インストールから最初の起動までを説明します。」 |
| 0:30-1:30 | ターミナル | 「まず、ダウンロードしたフォルダを開いて、インストーラーを実行します。」→ `./installer/install-mac.sh` を実行する画面。Docker, Node.js のチェックが通る様子。 |
| 1:30-2:30 | Docker起動画面 | 「Neo4j データベースが起動します。初回はイメージのダウンロードに数分かかります。」→ docker compose up の様子 |
| 2:30-3:30 | ブラウザ | 「http://localhost:7474 を開くと、データベースの管理画面が表示されます。」→ Neo4j Browser の画面 |
| 3:30-4:30 | Claude Desktop | 「Claude Desktop を再起動すると、ツール一覧に neo4j が表示されます。これで準備完了です。」→ MCP確認 |
| 4:30-5:00 | Claude Desktop | 「試しに『データベースの統計情報を教えて』と話しかけてみましょう。」→ 応答が返る画面 |

## 撮影のポイント

1. 画面収録ソフト: macOS の「QuickTime Player」→「新規画面収録」で十分
2. 解像度: 1920×1080 推奨
3. フォントサイズ: Claude Desktop のフォントを大きめに設定
4. デモデータ: 撮影前に必ず `./installer/load-demo-data.sh` を実行
5. 個人情報: 撮影にはデモデータのみを使用し、実際の個人情報は絶対に映さないこと
